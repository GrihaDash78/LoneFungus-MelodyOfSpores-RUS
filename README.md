# Lone Fungus: Melody of Spores на русском
Неофициальный русский перевод для [LoneFungus: Melody of Spores](https://store.steampowered.com/app/2463220/Lone_Fungus_Melody_of_Spores/) (BASTI GAMES, 2025)
> Перевод выполнен с помощью [DeepSeek](https://deepseek.com/) и последующими ручными исправлениями.
## 📑 Подробнее о структуре
**Переведено мною:**
- menus - интерфейс главного меню, паузы, и подсказок к управлению
- mapmakers - метки на карте и названия астральных врат
- descriptions - инвентарь, описания экипировки и остальных предметов

**Переведено с помощью DeepSeek:**
- intro - вступительное интро. <sub>Мелкие правки</sub>
- enemylog - бестиарий. <sub>Есть правки</sub>
- questlog - журнал квестов. <sub>Есть правки</sub>
- dialogue - диалоги. <sub>Куча правок ~и отсебятины~</sub>

## 📥 Установка
1. Скачайте архив **LanguageFiles** из [последнего релиза](https://github.com/GrihaDash78/LoneFungus-MelodyOfSpores-RUS/releases/latest)
2. Откройте корневую папку с игрой
   - Для Steam: ПКМ по названию игры -> Управление -> Просмотреть локальные файлы
3. Скопируйте все файлы с расширением _.ini_ из архива в папку _Language Files_
4. В корневой папке игры откройте файл _SETTINGS.ini_ любым текстовым редактором и замените самую последнюю строку на **language=russian**
## 🗑 Удаление
- Для Steam: ПКМ по названию игры -> Управление -> Свойства -> Установленные файлы -> Проверить целостность файлов игры
- Или же для смены языка по умолчанию: В корневой папке откройте файл _SETTINGS.ini_ любым текстовым редактором и замените самую последнюю строку на **language=english**
## 🐛 Известные проблемы
- Некоторые фразы могут звучать неестественно из-за ИИ-перевода
- Пропадает возможность сменить язык в главном меню после установки перевода
- В диалогах имена говорящих не изменены, т.к. в ином случае это приводит к сбросу портретов на дефолтный
